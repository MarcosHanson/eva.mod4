DESCRIPCION DEL SISTEMA

Este sistema de información está realizado para la mejora de gestión, control,
seguridad y disponibilidad de información de una empresa de asesorías de prevención
de riesgos.
 
Contiene módulos de gestión de información de los clientes y de la empresa
que provee el servicio, a través de un conjunto de páginas web.
 

 
----------------------------------------------------------------------------------

INSTRUCCIONES DE INSTALACION Y USO

Debe abrir el proyecto desde Eclipse EE y tener la previa instalación de Tomcat v9 y JAVA 8. 

Al ejecutar el proyecto obligatoriamente debe realizar el LOGIN para acceder al sistema (Usuario: rut válido / Contraseña: 1234).
La primera pagina luego del LOGIN es inicio que contiene un menú de navegación con las siguientes páginas:

-Inicio: Pagina principal luego del LOGIN.

-Contacto: Formulario de contacto para realizar consultas. 

-Crear Capacitación: Formulario para crear una capacitación en el sistema. 

-Listar Capacitación: Listado de capacitaciones registradas. 

-Login: Pagina para iniciar sesión.

-Listado de Usuario: Listado de Usuarios existentes y permite editar el registro de cada tipo de usuario. 

-Crear Usuarios: Crear Nuevo Usuario al sistema.

-Logout: Cierre de sesión.
